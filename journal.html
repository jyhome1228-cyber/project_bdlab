BDLab은 브랜드, 디자인, 지속가능성, 스케일업, 글로컬 전략에 대한 연구와 관점을 글로 기록합니다. 홈페이지에는 블로그형 글로, 인스타그램에는 간략한 카드형 콘텐츠로 확장합니다.

이곳의 글은 브랜드를 만드는 방식보다, 브랜드가 오래 작동하기 위해 필요한 구조와 기준을 다룹니다.

브랜드는 만들 때보다 운영될 때 더 많이 무너진다

Journal / Brand System

브랜드가 오래 지속되기 위해 필요한 운영 기준을 다룹니다.

좋은 로고보다 먼저 정리해야 할 것들

Journal / Identity

로고 이전에 필요한 브랜드의 철학과 구조를 정리합니다.

브랜드의 지속가능성은 오래 작동하는 구조에서 시작된다

Journal / Sustainability

지속가능한 브랜드를 정체성과 시스템 관점에서 해석합니다.

브랜드가 성장할수록 시각 자산은 기준이 되어야 한다

Journal / Visual Asset

성장 단계 브랜드에 필요한 시각 자산의 기준을 다룹니다.

작은 브랜드일수록 먼저 기준을 정리해야 한다

Journal / Small Brand

초기 브랜드가 먼저 갖춰야 할 언어와 시각 체계를 정리합니다.

브랜드는 하나의 장면이 아니라 이어지는 경험으로 기억된다

Journal / Experience

브랜드 경험이 여러 접점에서 연결되는 방식을 설명합니다.

브랜드 리뉴얼은 새로워 보이기 위해서만 하는 일이 아니다

Journal / Renewal

리뉴얼이 필요한 시점과 기준을 브랜드 구조 관점에서 봅니다.

브랜드의 첫인상은 하나의 로고가 아니라 여러 장면에서 결정된다

Journal / First Impression

컬러, 이미지, 말투, 웹사이트가 만드는 첫인상을 다룹니다.

패키지는 브랜드가 담기는 가장 작은 공간이다

Journal / Package

패키지를 단순 포장이 아닌 브랜드 공간으로 바라봅니다.

브랜드 컬러는 예쁜 색이 아니라 인식의 기준이다

Journal / Color

컬러가 브랜드 인식과 일관성에 미치는 역할을 설명합니다.

브랜드 가이드라인은 브랜드를 오래 운영하기 위한 기준이다

Journal / Guideline

브랜드 기준 문서가 필요한 이유를 운영 관점에서 설명합니다.

웹사이트는 브랜드가 어떤 구조로 작동하는지 보여준다

Journal / Web Identity

홈페이지를 브랜드 구조와 신뢰도의 매체로 해석합니다.

좋은 포트폴리오는 결과보다 과정을 먼저 보여준다

Journal / Portfolio

디자인 결과물보다 리서치와 전략의 흐름을 보여주는 방식.

브랜드 이미지 스타일은 보이는 방식을 정하는 기준이다

Journal / Image Style

사진, 렌더, 그래픽 이미지의 방향성을 정리하는 방법.

콘텐츠 디자인은 쌓일수록 같은 브랜드로 보여야 한다

Journal / Contents

SNS, 배너, 상세페이지가 같은 언어로 축적되는 기준.
